# HTML HOME
HTML is the standard markup language for Web pages. We can create  Website.

# HTML Introduction
HTML stands for Hyper Text Markup Language.

# HTML Editor
Web pages can be created and modified by using professional HTML editors Example: Notepad

# HTML Basics
HTML documents must start with a document type declaration <!DOCTYPE html> and it had <html>,<head>,<body> ,headings, link, images etc.

# HTML Element
HTML element is everything from the start tag to the end tag example <h1></h1>, <p></p>

# HTML Attributes
Attributes provide additional information about elements like name=”value”

# HTML Headings
HTML headings are titles or subtitles that you want to display on a webpage.

# HTML Paragraphs
A paragraph always starts on a new line, and is usually a block of text. The HTML <p> element defines a paragraph.

# HTML Style
The HTML style attribute is used to add styles to an element, such as color, font, size, and more.

# HTML text Formatting
HTML contains several elements for defining text with a special meaning. Example: <b>, <i>, <em> ,<strong> etc

# HTML Quotation
Examples : <blockquote>,<q>, <abbr>, <address>, <cite>, and <bdo> HTML elements.

# HTML Comment
HTML comments are not displayed in the browser, but they can help document your HTML source code.

# HTML Colors
HTML colors are specified with predefined color names, or with RGB, HEX, HSL, RGBA, or HSLA values.

# HTML style-CSS
Cascading Style Sheets (CSS) is used to format the layout of a webpage.

# HTML link
Links are found in nearly all web pages. Links allow users to click their way from page to page.

# HTML Images
Images can improve the design and the appearance of a web page.

# HTML Favicon
A favicon is a small image displayed next to the page title in the browser tab.

# HTML Page Title
The <title> element adds a title to your page.

# HTML tables
HTML tables allow web developers to arrange data into rows and columns.

# HTML List
HTML lists allow web developers to group a set of related items in lists.

# HTML block and inline
A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.

# HTML div
The <div> element is used as a container for other HTML elements.

# HTML Class
The class attribute is often used to point to a class name in a style sheet. It can also be used by JavaScript to access and manipulate elements with the specific class name. Multiple HTML elements can share the same class.

# HTML Id
The HTML id attribute is used to specify a unique id for an HTML element.You cannot have more than one element with the same id in an HTML document.

# HTML Buttons
Buttons let users interact with a web page. They can submit forms, run JavaScript, or trigger different actions when clicked.

# HTML iframe
The HTML <iframe> tag specifies an inline frame.An inline frame is used to embed another document within the current HTML document.

# HTML Javascript
JavaScript makes HTML pages more dynamic and interactive.

# HTML File path
A file path describes the location of a file in a web site's folder structure.

# HTML Head
The HTML <head> element is a container for the following elements: <title>, <style>, <meta>, <link>, <script>, and <base>.

# HTML Layout
Websites often display content in multiple columns. HTML has several semantic elements that define the different parts of a web page. Example <header>,<nav>, <footer> etc.

# HTML Responsive
Responsive Web Design is about using HTML and CSS to automatically resize, hide, shrink, or enlarge, a website, to make it look good on all devices (desktops, tablets, and phones).

# HTML Computer Code
HTML contains several elements for defining user input and computer code.

# HTML Semantic
A semantic element clearly describes its meaning to both the browser and the developer.

# HTML style Guide
Consistent, clean, and tidy HTML code makes it easier for others to read and understand your code.

# HTML Entities
Entity names or entity numbers can be used to display reserved HTML characters.

# HTML Symbols
Symbols or letters that are not present on your keyboard can be added to HTML using entities.

# HTML Emoji
Emojis look like images, but they are not. Emojis are letters (characters) from the UTF-8 (Unicode) character set.

# HTML Charset
To display an HTML page correctly, a web browser must know which character set to use.The character set is specified in the <meta> tag.

# HTML URL Encoder
Web browsers request pages from web servers by using a URL. A Uniform Resource Locator (URL) is used to address a document (or other data) on the web.

# HTML Versus XHTML
XML is a markup language where all documents must be marked up correctly. 

# HTML Forms
An HTML form is used to collect user input. The user input is most often sent to a server for processing.

# HTML Form Attribute
he <form> element in HTML is used to collect user input and send it to a server for processing. Important attributes like action, method, and target control where the data is sent, how it is sent (GET or POST), and where the response is displayed. Additional attributes such as autocomplete and novalidate help manage form behavior and validation.

# HTML Form element
HTML forms contain different elements like <input>, <label>, <select>, <textarea>, and <button> to collect user data in various formats. Elements such as <fieldset> and <legend> group related inputs, while <datalist> and <output> provide predefined options and display calculation results.

# HTML Input type
The <input> element in HTML supports many different types, such as text, password, email, radio, checkbox, date, and more, to collect various kinds of user data. Each input type serves a specific purpose, and if no type is specified, the default value is text.

# HTML Input Attribute
The default value of the HTML <input> element’s size attribute is 20. This means that if you do not specify the size attribute, the input field will display with a width of 20 characters by default.

# Input Form Attibutes
formtype is not a legal attribute for the HTML <input> element. Valid form-related attributes include formaction, formmethod, formenctype, formtarget, and formnovalidate. Since formtype is not defined in the HTML specification, browsers do not recognize it.

# HTML Canvas
The HTML <canvas> element is used to draw graphics on a web page using JavaScript. It acts as a container for graphics, and you must use JavaScript methods to draw shapes, text, images, and animations inside it. Canvas is supported by all major browsers and is commonly used for games, charts, and dynamic visual content.

# HTML SVG
SVG (Scalable Vector Graphics) is an XML-based format used to create vector graphics for the web. It is resolution independent, so graphics do not lose quality when resized. SVG elements can also be styled, animated, and controlled using CSS and JavaScript.

# HTML Media
HTML multimedia includes audio, video, music, and animations displayed on web pages. Modern HTML supports formats like MP4, WebM, and Ogg for video, and MP3, WAV, and Ogg for audio. These can be added using the <video> and <audio> elements.

# HTML Video
The HTML <video> element is used to display videos on a web page. It supports formats like MP4, WebM, and Ogg, and includes attributes such as controls, autoplay, and muted to manage playback. The <source> tag allows multiple video formats so the browser can choose a supported one.

# HTML Audio
The HTML <audio> element is used to play sound files on a web page. It supports formats like MP3, WAV, and Ogg, and uses the controls attribute to display play and volume options. The <source> tag allows multiple audio formats so the browser can choose a supported one.

# HTML Plugins
HTML plug-ins were used to extend browser features, such as running Java applets or Flash content. However, modern browsers no longer support most traditional plug-ins due to security reasons. Today, <object> and <embed> are mainly used to display media or external content in web pages.

# HTML Youtube 
You can easily embed a YouTube video in HTML using the <iframe> element and the video’s unique ID. Set the src attribute to https://www.youtube.com/embed/VIDEO_ID and define the width and height. You can also add parameters like autoplay=1, mute=1, loop=1, or controls=0 to customize playback.
